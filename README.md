## Development Setup

### Setting Up a Project

Install the Angular CLI globally:

```
cd project
npm install
```

Run the application:

```
ng serve
```
or

```
npm start
```

## Bugs in the project
### No image displayed
1 If no image appears when you search in the search bar, you need to remove the content of the search field or change any option in the filter.

2 If there is nothing after the first point, check your Flickr key.
